# Media Feed API

The Media Feed API provides structured ecosystem content to external applications and dashboards.

## Endpoint

`GET /v1/feed`

## Purpose

This endpoint supports:
- reference mobile applications
- partner application interfaces
- public-sector or institutional dashboards

## Example filters

- source
- category
- language
- limit
- cursor

## Role in the platform

This endpoint is not only a content feature.

It also serves as part of the external platform story by showing that Namazu API can deliver structured ecosystem information to third-party systems.
