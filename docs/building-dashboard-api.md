# Building Dashboard API

The Building Dashboard API provides retrieval of building-linked workflow status or latest result information.

## Endpoint

`GET /v1/buildings/{building_id}/results`

## Purpose

This endpoint supports:
- building-linked workflow confirmation
- latest result retrieval
- dashboard integration
- future analytical expansion

## Role in the platform

This endpoint helps position Namazu API as a building-oriented platform rather than only a raw data intake interface.

That distinction is important in institutional and enterprise discussions.
