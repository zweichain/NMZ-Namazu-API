# ZweiMedia iOS Reference Client

ZweiMedia is the first reference mobile client for the Namazu ecosystem.

## Role in the ecosystem

ZweiMedia demonstrates two important directions:

1. consumption of curated ecosystem media
2. contribution of smartphone sensor data into the Namazu platform

## Why this matters

Namazu API is not being introduced as a theoretical interface.

It is grounded in an actual mobile-client direction that already combines:
- media
- user-facing app flow
- smartphone device capabilities
- cloud-linked platform behavior

## Current reference concept

The reference client concept includes:

- article / media retrieval
- presentation of ecosystem content
- device-side motion / acceleration capture
- upload of sensor data to a cloud layer
- future linkage to building dashboards and structural analysis workflows

## Recommended future additions

As the API matures, ZweiMedia or a successor reference client can demonstrate:

- authenticated API usage
- feed filtering
- upload status tracking
- building-linked result retrieval
- future contribution / reward logic
- future TGL-linked result display

## Positioning note

ZweiMedia should be presented as the first reference client, not as the entire Namazu platform.

The broader platform includes:
- Namazu API
- Building Dashboard
- future TGL analysis
- contribution / verification logic
- partner and institutional integrations
